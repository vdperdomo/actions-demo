Methods to use:


***register: To register a new action. (will not be exposed to the customer)

@param {string} actionName - action name
@param {function | Object} action - function to trigger or action object

register(actionName, action)


***addInterceptor: To register an interceptor for an action (will be exposed to the customer).

@param {string} actionName - Action name
@param {function} callback - Interceptor method
@param {Boolean} once - option to trigger the interceptor just one time


addInterceptor(actionName, callback, once)

The callback is an async method. Within it, it should make the call to continue() in case its business logic allows to continue with the workflow. Otherwise it should call to intercept(). If it doesn't, exist a timeout mecanish that returns an error after 5000ms


***remove: To remove an existing action

@param {string} actionName - action name

remove(actionName)


***removeInterceptor: To remove an interceptor for an especific action

@param {string} actionName - action name
@param {function} callback - Interceptor method to remove

removeInterceptor(actionName, callback)
